# Metasploitable
Un repositorio explicando como he atacado algunas de las vulnerabilidades de la maquina virutal Metasploitable.

## ⚠️ Disclaimer
Este repositorio documenta ataques realizados EXCLUSIVAMENTE contra 
una máquina virtual Metasploitable en un laboratorio aislado y 
controlado (red interna, sin conexión a redes externas). Esta 
información tiene fines exclusivamente educativos. No uses estas 
técnicas contra sistemas sin autorización explícita.
